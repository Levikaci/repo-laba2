ПРИЛОЖЕНИЕ ДЛЯ РАБОТЫ С ХИМИЧЕСКИМИ ДАННЫМИ Программное обеспечение для моделирования химических реакций, управления лабораторными данными и визуализации молекулярных структур. Приложение предназначено для студентов, исследователей и лаборантов. Позволяет хранить справочные данные, проводить расчеты стехиометрии и вести электронный лабораторный журнал.

ФУНКЦИОНАЛЬНОСТЬ Справочник химических элементов и соединений Визуализация 3D молекулярных структур Автоматическая балансировка химических уравнений Расчет молярных масс и стехиометрических коэффициентов Ведение электронного лабораторного журнала Поиск по базе реакций и веществ Экспорт отчетов в форматах CSV, JSON, PDF

ТРЕБОВАНИЯ Операционная система: Windows 10/11, macOS 12+, Linux (Ubuntu 20.04+) PostgreSQL 13+ Оперативная память: не менее 4 ГБ Свободное место на диске: 2 ГБ Графический процессор с поддержкой OpenGL 3.3 (для 3D визуализации)
