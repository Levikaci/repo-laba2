
# ПРИЛОЖЕНИЕ ДЛЯ РАБОТЫ С ХИМИЧЕСКИМИ ДАННЫМИ

## 1. Описание проекта
Программное обеспечение для моделирования химических реакций, управления лабораторными данными и визуализации молекулярных структур. Приложение предназначено для студентов, исследователей и лаборантов. Позволяет хранить справочные данные, проводить расчеты стехиометрии и вести электронный лабораторный журнал.

## Функциональность
Отображаемые данные
Поиск и фильтрация
Сортировка
Все операции выполняются в рамках единой табличной структуры
Система содержит не менее 200 записей тестовых данных
Поддержка экспорта данных (CSV/Excel) — опционально

## Архитектура проекта
Тип архитектуры: Клиент-сервер 
**Технологии:**
C#
PostgresSQL

## ТРЕБОВАНИЯ
Операционная система: Windows 10/11, macOS 12+, Linux (Ubuntu 20.04+)
PostgreSQL 13+
Оперативная память: не менее 4 ГБ

## Использование
После запуска приложения пользователь видит:
Таблица данных с колонками: ФИО | Инв. номер | Тип | Стоимость | Место
Панель управления:
Поисковая строка (поддержка подстрок)
